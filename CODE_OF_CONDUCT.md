# Code of Conduct

This project expects respectful, constructive collaboration.

## Expected Behavior

- Be respectful and professional.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask for clarification before escalation.

## Unacceptable Behavior

- Harassment, threats, or discrimination.
- Personal attacks or abusive language.
- Sharing private information without consent.

## Enforcement

Maintainers may remove comments, reject contributions, or restrict participation
for behavior that violates this policy.
